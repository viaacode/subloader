AVO Subtitle Uploader

python flask app om subtitles snel online te krijgen. Vanilla javascript voor ajax calls en injecteren flow player... Biobaner logt in met oas.viaa.be (zien of ik account heb enz.) on prem openshift. mediahaven v2 api. (oauth v2 voor de rest calls van de app zelf).

Form : pid -> geef je in , dan haal je het op uit MediaHaven en krijgt data /metadata terug: dan zie je: titel omschrijving createdatum

je toont video met flowplayer op deze page metadata. en kan je een subtitle file uploaden.

effectief submit button die gaat dan met ajax de file zelf uploaden. dit is ook een mediahaven rest-api call die het bestand zelf er zet. srt file komt zo op juiste plaats op testbeeld en de rechten (later is er nog een redactie waarbij iemand iets aanvingt om hem online te zetten).
